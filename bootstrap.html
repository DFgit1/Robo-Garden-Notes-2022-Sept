IMPORTANT

  • Width usually is determined by parent

There are 4 main display behaviors (Types of rendering box)

  • display: none;

    • Elements will not be rendered (but still exist in the DOM)

  • display: inline;

    • Elements will not start on a new line (No line break before element)
    • Elements will only take as much width as necessary (Width depends on content)
    • Width and Height properties will have no effect
    • Default for elements <span>,<a>,<img>

  • display: block;

    • Element will always start on a new line (Line break comes before element)
    • Element takes up all available width (Inherit from parents, Default HTML behavior)
    • Element takes up as little height as possible to show content (Inherit from child, Default HTML behavior)
    • Width and Height properties can be set
    • <div>,<h1>-<h6>,<p>,<form>,<header>,<footer>,<section>

  • display: inline-block;

    • Element will not start on new line (Just like display: inline;)
    • Element will only take as much width as necessary (Just like display: inline;)
    • Width and Height properties can be set. (Just like display: block;)
    • No elements have this display type by default [1]
