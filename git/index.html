GIT Notes

Intro

What is git?

git is versioning control software

What choices do I have beside git?

SVN is the second most popular versioning control system

Terminology

  • Repository a collection of code that is being version controlled by git
  • Staging Area / Staged Files staged files are ready to be commited, but not yet commited to version history
  • Working tree Files we are currently working on
  • HEAD Tip of our branch

Git Commands

Command Action Preformed Notes
git init initializes our git repository in the current working directory
git status Show us what has been changed in the current working tree
git add <name-of-file-or-files> Mark files to be staged
git commit -m '<commit message>' Commit staged changes to local repository
git push Syncing local commits to upstream(origin/authoritative) repo
